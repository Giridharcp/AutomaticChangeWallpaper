# AutomaticChangeWallpaper
Simple python script to change your wallpaper automatically based on the keyword. if you type in a Keyword then the script is going to automatically download in HD image of the one of the images related to the keyword and then save it in your PC and set it as your wallpaper.
import the library google_images_download using pip 
If you prefer change the destination where your images are saved.
